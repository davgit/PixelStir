PixelStir
=========

PixelStir is a Facebook application for bulk photo uploading. 

Features
--------

* Bulk upload.
* Drag and drop.
* Image filters.
* Image cropping.

Technical Details
-----------------

* Pure JavaScript implementation.
* Filters via [CamanJS](http://camanjs.com/).
* Canvas based client side resizing pre-upload.
